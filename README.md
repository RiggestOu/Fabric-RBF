##Fabric-RBF
A Radial Basis Interpolation solver in KL.

* A few math utils.
* A N*M Matrix class.
* A RBF solver.
* A Maya UI to interface with the solver.

###The Matrix Class

Features:
* LU decomposition
* SVD decomposition
* Inversion
* Determinant
* Multiplications
* Transpose
* Linear System Solving
